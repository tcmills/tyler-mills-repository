These are command line games that I have made in my personal time in Python.

Battleship.py is based off of the Battleship boardgame.
Blackjack.py is based off of the official Blackjack rules.
Letters.py is based off of the Love Letter card game designed by Seiji Kanai and published by AEG and Z-Man Games.
MH Calc.py is not a game but a calculater using probability to predict drop rates for materials in Monster Hunter Rise.
Mancala.py is based off of the Mancala boardgame.
RockPaperScissors.py is based off of Rock, Paper, Scissors.

Unearthed.py is a game I developed.
